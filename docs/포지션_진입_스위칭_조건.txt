
📊 포지션 진입 및 롱/숏 스위칭 조건 정리


🎯 자동 트레이딩 시스템 포지션 진입 조건


📋 기본 진입 조건
• 최소 신뢰도: 80% 이상
• 최소 거래량 비율: 1.2배 이상  
• 최소 보유 시간: 30분
• 최대 손실 제한: -5% (스위칭 허용)


🔥 롱(LONG) 포지션 진입 조건


📊 4개 조건 중 3개 이상 만족해야 함:

1. 상승 트렌드 확인
   - 조건: SMA20 > SMA50
   - 설명: 단기 이동평균이 중기 이동평균을 상향 돌파

2. RSI 건전성 확인  
   - 조건: 40 < RSI < 70
   - 설명: 과매수/과매도가 아닌 건전한 모멘텀 구간

3. 거래량 지지 확인
   - 조건: 거래량 비율 > 1.2
   - 설명: 평균 대비 20% 이상 거래량 증가로 상승 지지

4. 골든크로스 확인
   - 조건: EMA12 > EMA26
   - 설명: 단기 지수이동평균이 장기 지수이동평균 상향 돌파


⚡ 숏(SHORT) 포지션 진입 조건  


📊 4개 조건 중 3개 이상 만족해야 함:

1. 하락 트렌드 확인
   - 조건: SMA20 < SMA50
   - 설명: 단기 이동평균이 중기 이동평균을 하향 돌파

2. RSI 과매수 확인
   - 조건: RSI > 70
   - 설명: 과매수 구간에서 하락 전환 예상

3. 거래량 지지 확인
   - 조건: 거래량 비율 > 1.2
   - 설명: 평균 대비 20% 이상 거래량 증가로 하락 지지

4. 데드크로스 확인
   - 조건: EMA12 < EMA26
   - 설명: 단기 지수이동평균이 장기 지수이동평균 하향 돌파


🔄 롱/숏 스위칭 조건


📊 롱 → 숏 스위칭 조건
3개 조건 중 2개 이상 만족해야 함:

1. 가격 하락 확인
   - 조건: 현재가 < SMA20
   - 설명: 현재가가 단기 이동평균선 아래로 하락

2. RSI 과매수 확인
   - 조건: RSI > 70
   - 설명: 과매수 구간에서 하락 전환 신호

3. 데드크로스 확인
   - 조건: EMA12 < EMA26
   - 설명: 단기 지수이동평균이 장기 지수이동평균 하향 돌파

📊 숏 → 롱 스위칭 조건
3개 조건 중 2개 이상 만족해야 함:

1. 가격 상승 확인
   - 조건: 현재가 > SMA20
   - 설명: 현재가가 단기 이동평균선 위로 상승

2. RSI 과매도 확인
   - 조건: RSI < 30
   - 설명: 과매도 구간에서 반등 전환 신호

3. 골든크로스 확인
   - 조건: EMA12 > EMA26
   - 설명: 단기 지수이동평균이 장기 지수이동평균 상향 돌파


📈 기술적 분석 전략별 진입 조건


📊 데이 트레이딩 전략 (15분-1시간 단위)
매수 조건:
• 단기 상승추세 (SMA10 > SMA20)
• 가격이 단기 평균선 위 위치
• RSI 건전한 모멘텀 구간 (45-75)
• MACD 긍정적 신호
• 거래량 증가 지지
• 볼린저밴드 적정 위치

목표 수익: 2.5%
손절: SMA20 아래 1.5%

🌊 스윙 트레이딩 전략 (며칠-몇 주 단위)
매수 조건:
• 상승 트렌드 확인 (SMA20 > SMA50)
• 조정 완료 후 재상승 시작
• MACD 골든크로스 확인
• RSI 건전한 수준 (40-70)
• 볼린저밴드 중간 위치 (안전 구간)

목표 수익: 8%
손절: SMA50 아래 2%

📊 포지션 트레이딩 전략 (몇 주-몇 달 단위)
매수 조건:
• 장기 상승 트렌드 확인 (SMA50 > SMA200)
• 가격이 주요 이동평균선 위 위치
• MACD 강세 신호 (MACD > 0)
• RSI 강세 구간 (> 50)
• 골든크로스 발생 (SMA50 > SMA200)

목표 수익: 25%
손절: SMA200 아래 5%

🔄 평균 회귀 전략
매수 조건 (극도 과매도):
• 볼린저 %B < 0.1 (볼린저 하단 근처)
• RSI < 25 (극도 과매도)
• 가격이 MA 아래 2%

매도 조건 (극도 과매수):
• 볼린저 %B > 0.9 (볼린저 상단 근처)
• RSI > 75 (극도 과매수)
• 가격이 MA 위 2%


⚠️ 안전 장치 및 리스크 관리


🛡️ 손실 제한
• -5% 이상 손실 시 스위칭 차단
• 손실이 과도할 때는 스위칭 대신 손절 고려

⏰ 최소 보유 시간
• 30분 미만 보유 시 스위칭 금지
• 단기 변동성에 의한 불필요한 스위칭 방지

🎯 신뢰도 검증
• 80% 미만 신호는 무시
• 신뢰도가 낮은 신호로 인한 손실 방지

📊 거래량 확인
• 거래량 부족 시 진입 보류
• 유동성 부족으로 인한 슬리피지 방지

💰 포지션 크기 관리
• 계좌 대비 포지션 크기: 2%
• 과도한 레버리지 사용 방지


📝 참고사항


• 모든 조건이 만족되어야 실제 포지션 진입/스위칭 실행
• 기술적 지표는 15분봉 기준으로 계산
• 실시간 시장 상황에 따라 조건이 동적으로 조정될 수 있음
• 과거 성과가 미래 수익을 보장하지 않음
• 충분한 테스트 후 실제 거래에 적용 권장
